| 3  | Park | Sales  | 3200   |
| 4  | Choi | Dev    | 3800   |
| 5  | Jung | HR     | 2500   |
| 6  | Kang | Dev    | 4500   |

```sql
SELECT dept, COUNT(*) AS cnt, SUM(salary) AS total
FROM emp
WHERE salary >= 3000
GROUP BY dept
HAVING COUNT(*) >= 2
ORDER BY total DESC;
```

## 정답 및 해설

### 문제 1 정답: `4 8 6 6`

```
p = a + 1 -> p는 a[1]을 가리킴
*p       = a[1] = 4
*(p+2)   = a[3] = 8
*p + 2   = a[1] + 2 = 4 + 2 = 6  (연산자 우선순위: *p가 먼저!)
p++      -> p는 a[2]를 가리킴
*p       = a[2] = 6
```

### 문제 2 정답: `200 100`

```
Parent obj = new Child()
obj.display() -> 오버라이딩 -> Child.display() -> 200 출력
obj.x -> 변수는 참조 타입(Parent) 기준 -> Parent.x = 100
```
핵심: 메서드는 자식 것, 변수는 부모 것!

### 문제 3 정답:
```
[3, 4, 5, 6]
[1, 4, 7]
```
- `a[2:6]` = 인덱스 2,3,4,5 -> [3,4,5,6]
- `a[::3]` = 0번부터 3칸씩 -> a[0]=1, a[3]=4, a[6]=7

### 문제 4 정답: `13`

```
f(n<=1) = 1 (주의: f(0)=1, f(1)=1)
f(2) = f(1)+f(0) = 1+1 = 2
f(3) = f(2)+f(1) = 2+1 = 3
f(4) = f(3)+f(2) = 3+2 = 5
f(5) = f(4)+f(3) = 5+3 = 8
f(6) = f(5)+f(4) = 8+5 = 13
```

### 문제 8 정답: `25`

```
대각선1 (i,i):    a[0][0]+a[1][1]+a[2][2] = 1+5+9 = 15

sum = 15 + 15 - 5 = 25
```

### 문제 10 정답: `A2 A1 B2 B1 5`

```

-> 출력 "B1 "
obj.getX() -> A 소속 메서드 -> return A.x = 5
```
