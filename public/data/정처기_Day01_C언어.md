## PART 1: 포인터 핵심 정리

### 1-1. 포인터 기본 개념

```c
int a = 10;
int *p = &a;    // p는 a의 주소를 저장

printf("%d", a);    // 10      (변수의 값)
printf("%p", &a);   // 0x7fff  (변수의 주소)
printf("%p", p);    // 0x7fff  (포인터가 저장한 주소)
printf("%d", *p);   // 10      (포인터가 가리키는 값 = 역참조)
```

### 1-2. 포인터 연산 핵심 공식

```
p       → 주소값
*p      → 해당 주소의 값 (역참조)
p + 1   → 다음 요소의 주소 (자료형 크기만큼 이동)
*(p+1)  → 다음 요소의 값
&a      → 변수 a의 주소
```

### 1-3. 문자열 포인터 (시험 최빈출!)

```c
char *p = "KOREA";
// 메모리:  K  O  R  E  A  \0
// 인덱스:  0  1  2  3  4

printf("%s", p);      // KOREA    (%s: 해당 주소부터 \0까지 출력)
printf("%s", p+3);    // EA       (3번째부터 끝까지)
printf("%c", *p);     // K        (첫 글자)
printf("%c", *(p+3)); // E        (3번째 글자)
printf("%c", *p+2);   // M        ('K'=75, 75+2=77='M')
```

> **핵심**: `*p+2`는 값에 2를 더함, `*(p+2)`는 주소를 2 이동 후 값을 가져옴

### 1-4. 포인터와 배열의 관계

```c
int arr[] = {10, 20, 30, 40, 50};
int *p = arr;    // 배열명 = 첫 번째 요소의 주소

// 아래는 모두 동일한 결과:
arr[2]    == *(arr+2)   == *(p+2)   == p[2]   // → 30
&arr[2]   == arr+2      == p+2                 // → 30의 주소
```

## PART 2: 배열 핵심 정리

### 2-1. 2차원 배열

```c
int arr[2][3] = {1, 2, 3, 4, 5, 6};

// 메모리 구조:
// arr[0][0]=1  arr[0][1]=2  arr[0][2]=3
// arr[1][0]=4  arr[1][1]=5  arr[1][2]=6

printf("%d", arr[1][2]);    // 6
printf("%d", *(arr[0]+1));  // 2  (arr[0]은 1행의 시작주소)
```

### 2-2. 배열 포인터 vs 포인터 배열

```c
int (*p)[3];    // 배열 포인터: 3개짜리 배열을 가리키는 포인터
int *p[3];      // 포인터 배열: 포인터 3개를 저장하는 배열
```

## PART 3: 구조체 핵심 정리

### 3-1. 구조체 선언과 접근

```c
struct Student {
    char name[20];
    int score;
};

struct Student s1 = {"Kim", 90};
struct Student *p = &s1;

// 접근 방법 2가지:
printf("%s", s1.name);    // 변수로 접근: .  (dot)
printf("%d", p->score);   // 포인터로 접근: -> (arrow)
```

> **시험 핵심**: `.` vs `->` 구분!
> - 구조체 변수: `변수.멤버`
> - 구조체 포인터: `포인터->멤버`

### 3-2. 구조체 배열

```c
struct Student st[3] = {
    {"데이터1", 95, 88},
    {"데이터2", 84, 91},
    {"데이터3", 86, 75}
};
struct Student *p = &st[0];

// p     → st[0]
// p+1   → st[1]
// p+2   → st[2]
// (p+1)->score → st[1].score → 84
```

## PART 4: 자주 나오는 함수/패턴

### 4-1. 재귀함수

```c
int f(int n) {
    if (n <= 1) return 1;
    return n * f(n-1);
}
// f(5) = 5 * 4 * 3 * 2 * 1 = 120 (팩토리얼)
```

### 4-2. swap 함수 (Call by Reference)

```c
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int x = 10, y = 20;
swap(&x, &y);   // x=20, y=10
```

### 4-3. 연산자 우선순위 (시험 함정!)

| 우선순위 | 연산자 | 설명 |
|----------|--------|------|
| 1 | `()` `[]` `->` `.` | 괄호, 배열, 멤버접근 |
| 2 | `!` `~` `++` `--` `*`(역참조) `&`(주소) | 단항 연산자 |
| 3 | `*` `/` `%` | 산술 (곱/나눗셈) |
| 4 | `+` `-` | 산술 (덧/뺄셈) |
| 5 | `<<` `>>` | 시프트 |
| 6 | `<` `<=` `>` `>=` | 관계 |
| 7 | `==` `!=` | 동등 |
| 8 | `&` `^` `\|` | 비트 AND, XOR, OR |
| 9 | `&&` `\|\|` | 논리 AND, OR |
| 10 | `? :` | 삼항 |
| 11 | `=` `+=` `-=` 등 | 대입 |

### 4-4. printf 서식 지정자

| 서식 | 의미 | 예시 |
|------|------|------|
| `%d` | 정수 (10진수) | `printf("%d", 10)` → 10 |
| `%c` | 문자 1개 | `printf("%c", 65)` → A |
| `%s` | 문자열 | `printf("%s", "Hi")` → Hi |
| `%f` | 실수 | `printf("%.2f", 3.14)` → 3.14 |
| `%o` | 8진수 | `printf("%o", 10)` → 12 |
| `%x` | 16진수 | `printf("%x", 255)` → ff |

### 문제 4 (스택 + 포인터) ★★★★★

```c
#include <stdio.h>
#define MAX 10

int stack[MAX];
int top = -1;

void push(int num) { stack[++top] = num; }
int pop() { return stack[top--]; }
int isEmpty() { return (top == -1) ? 1 : 0; }

int main() {
    push(5); push(2);
    while (!isEmpty()) {
        printf("%d", pop());
        push(4); push(1); printf("%d", pop());
        push(3); printf("%d", pop()); printf("%d", pop());
        push(6); printf("%d", pop()); printf("%d", pop());
    }
}
```

**트레이싱 메모:**
```

push(4) push(1) → 스택: [___, ___, ___]
pop() → 출력: ___ / 스택: [___, ___]
push(3) → 스택: [___, ___, ___]
```

### 문제 7 (포인터 + 증감 연산) ★★★★

```c
#include <stdio.h>
int main() {
    int a[] = {1, 2, 3, 4, 5};
    int *p = a;

    printf("%d ", *p++);
    printf("%d ", *++p);
    printf("%d ", ++*p);
    printf("%d ", (*p)++);
    printf("%d", *p);
}
```

**트레이싱 메모:**
```
*p++   → 먼저 *p(값) 출력, 그 후 p++ (주소 이동)
*++p   → 먼저 ++p (주소 이동), 그 후 *p (값)
++*p   → 먼저 *p (값), 그 값에 ++
(*p)++ → 먼저 *p (현재 값) 출력, 그 후 값++
*p     → 현재 값

p 위치:  a[0] → a[___] → a[___] → a[___] → a[___]
값:      ___    ___     ___     ___     ___

출력: _______________
```
