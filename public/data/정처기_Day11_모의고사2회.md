## 정답 및 해설

### 문제 1 정답: `O M EA`

```
p = "KOREA"  -> K(0) O(1) R(2) E(3) A(4)
*(p+1) = 'O'
*p+2  = 'K' + 2 = ASCII 75+2 = 77 = 'M'  (문자에 정수 더함!)
p+3   = "EA"  (포인터 이동 후 문자열 출력)
```

### 문제 2 정답: `22 1 22 22`

```
obj1 = A 타입, B 객체
obj1.fn() -> B.fn() -> B.n = 2+20 = 22, 출력 "22 "
obj1.n    -> 변수는 참조타입(A) -> A.n = 1, 출력 "1 "

obj2 = B 타입, B 객체
obj2.fn() -> B.fn() -> B.n = 2+20 = 22, 출력 "22 "
obj2.n    -> B.n = 22 (이미 변경됨), 출력 "22"
```
핵심: obj1.n은 A 타입이므로 A.n(=1), obj2.n은 B 타입이므로 B.n(=22)

### 문제 3 정답:
```
{'K', 'O', 'R'}
{'A', 'E'}
{'A', 'E', 'K', 'N', 'O', 'R'}
```
- `&` = 교집합, `-` = 차집합, `|` = 합집합

### 문제 4 정답: `20 35 15`

```
func(x, &y): a는 값 전달(x 안 변함), b는 주소 전달(y 변함)
a = 20+5 = 25 (지역변수, x에 반영 안됨)
*b = 30+5 = 35 (y에 반영됨)
g = 10+5 = 15  (전역변수 변경)
-> x=20, y=35, g=15
```

### 문제 5 정답: `P10 C C20`

```
new Child(20)
-> Child(int x) -> this() 호출
  -> Child() -> super(10) 호출
    -> Parent(10) -> 출력 "P10 "
  -> 출력 "C "
-> 출력 "C20 "
```
주의: this()를 먼저 호출하므로 Child()를 거쳐서 super(10)에 도달

### 문제 8 정답: `54321 0 0 0 0`

```
int n[5] = {54321};

{5,4,3,2,1}이 아님!
```
